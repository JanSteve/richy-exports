# Richy Exports — "The Grand Loom"

**Premium Yarn & Carpet Import/Export Website**

A World Expo-grade single-page website for Richy Exports, built with pure HTML, CSS, and vanilla JavaScript. Zero dependencies.

## Features

- **Live Loom Weave** — Animated SVG diamond-grid carpet pattern
- **Thread Loader** — Golden thread animation that weaves the brand identity
- **Two Division Panels** — Burgundy (Yarn) and emerald (Carpet) editorial halls
- **Product Catalogue** — 12 products with tab filtering and woven-border cards
- **Trade Route Map** — SVG world map with animated thread lines to 40+ countries
- **Process Timeline** — Scroll-animated horizontal thread with popping nodes
- **Gold Shimmer** — Flowing gradient on brand typography

open index.html
