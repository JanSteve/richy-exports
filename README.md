# RICHY EXPORTS — Premium Herbal Products Export Website

## Tech Stack
React 18 + Framer Motion + Tailwind CSS + Vite

## Local Development
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```

## Deploy
Hosted on Vercel. Auto-deploys from main branch.

## Environment Variables
Copy `.env.example` to `.env` and fill in values.
